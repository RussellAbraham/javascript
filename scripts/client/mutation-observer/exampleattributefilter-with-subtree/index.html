MutationObserver Example - attributeFilter with subtree

This page uses the MutationObserver API to watch for changes to the DOM tree. In this case, the MutationObserver is looking for specific HTML attributes to be changed on the targeted element, and the same attributes on the target's child elements.

A "Start..." and a "Stop..." button are used to start and stop the MutationObserver. Console messages are displayed specifying which change took place and whether the MutationObserver is started or stopped.

This paragraph element will be observed for changes to its attributes.
