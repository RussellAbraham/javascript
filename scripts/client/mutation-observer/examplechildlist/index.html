MutationObserver Example - childList

This page uses the MutationObserver API to watch for changes to the DOM tree. In this case, the MutationObserver is looking for child nodes to be changed on the targeted element (the list).

A "Start..." and a "Stop..." button are used to start and stop the MutationObserver. Console messages are displayed specifying which change took place and whether the MutationObserver is started or stopped.
