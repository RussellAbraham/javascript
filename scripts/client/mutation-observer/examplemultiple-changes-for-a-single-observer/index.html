MutationObserver Example - Multiple Changes for a Single Observer

This demo uses a single MutationObserver to observe DOM changes on two different targets. The one caveat here is that when you "stop" observing, the MutationObserver will stop observing for all specified targets.

  • Apples
  • Oranges
  • Bananas
  • Peaches

  • Dogs
  • Cats
  • Chickens
  • Horses
