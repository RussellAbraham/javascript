MutationObserver Example - characterData

This page uses the MutationObserver API to watch for changes to the DOM tree. In this case, the MutationObserver is looking for character data changes on the targeted element.

A "Start..." and a "Stop..." button are used to start and stop the MutationObserver. Console messages are displayed specifying which change took place and whether the MutationObserver is started or stopped.
